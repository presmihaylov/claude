# Claude Skills

A collection of reusable Claude Code skills.

## github-code-reviewer

High-signal code review skill for GitHub PRs. Reviews code with surgical precision, flagging only high-severity issues (bugs, security, performance, breaking changes) via succinct inline comments on specific lines.

### Features

- Surgical precision reviews focusing on critical issues only
- Inline comments on specific problematic lines
- Skips style, nits, and minor improvements
- High signal, low noise approach
- Multi-project compatible
